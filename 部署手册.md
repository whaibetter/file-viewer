# File Viewer 部署手册

## 环境要求

- Ubuntu/Debian 系统
- Python 3.8+
- Nginx
- systemd

## 快速部署

### 方式一：使用安装脚本（推荐）

```bash
# 1. 克隆或上传项目到服务器
cd /path/to/file-viewer

# 2. 运行安装脚本
sudo ./install.sh

# 3. 修改默认密码
sudo file-viewer passwd
```

### 方式二：手动部署

```bash
# 1. 安装依赖
pip install pyyaml flask flask-socketio

# 2. 复制文件
sudo cp file-viewer-server.py /usr/local/bin/
sudo cp file-viewer /usr/local/bin/
sudo chmod 755 /usr/local/bin/file-viewer
sudo cp index.html /usr/share/nginx/html/file-viewer/

# 3. 安装 systemd 服务
sudo cp file-viewer.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable file-viewer
sudo systemctl start file-viewer

# 4. 配置 Nginx（可选）
# 参考 README.md 中的 Nginx 配置
```

## 访问

- 地址：`http://<服务器IP>/file-viewer/`
- 默认密码：`admin`

## 管理命令

```bash
file-viewer start      # 启动服务
file-viewer stop       # 停止服务
file-viewer restart    # 重启服务
file-viewer status     # 查看状态
file-viewer passwd     # 修改密码
file-viewer logs       # 查看日志
```

## 目录结构

```
file-viewer/
├── config.yaml              # 配置文件
├── file-viewer              # 管理脚本
├── file-viewer-server.py    # 后端服务
├── file-viewer.service      # systemd 服务
├── index.html               # 前端页面
├── install.sh               # 安装脚本
├── README.md                # 说明文档
└── 部署手册.md              # 本文档
```

## 配置文件位置

- 系统配置：`/etc/file-viewer/config.yaml`
- 密码文件：`/etc/file-viewer/passwd`
- 删除白名单：`/etc/file-viewer-whitelist.json`

## 常见问题

### 服务无法启动

```bash
# 查看错误日志
sudo journalctl -u file-viewer -n 50

# 检查端口是否被占用
sudo lsof -i :9001
```

### 修改配置后不生效

```bash
sudo file-viewer restart
```

### 无法删除文件

检查该路径是否在删除白名单中，通过设置面板添加。
